# NARES - Neuro Adaptive Route Emergency System

**Project Abbreviation:** NARES  
**Description:**  
NARES is an intelligent navigation platform for neurodivergent individuals, offering real-time adaptive route suggestions, emergency assistance, and interactive guidance using AI and Google Maps. Made with ❤️ for neurological people, it ensures safe, calm, and efficient travel.

## Technology Stack
- **Framework:** Next.js (v15) with App Router  
- **Language:** TypeScript  
- **UI Library:** React (v18)  
- **Styling:** Tailwind CSS  
- **UI Components:** ShadCN/UI  
- **Backend & Database:** Firebase (Auth & Firestore)  
- **Generative AI:** Genkit  
- **Mapping:** Google Maps Platform  

## Features
- Adaptive route suggestions  
- Emergency alerts and location tracking  
- Calm mode for stress-free navigation  
- Gamified cognitive exercises & sensory-friendly features  
- Login for citizens and authorities  
- Fully responsive design  

## Quick Setup
1. Clone the repo:  
```bash
git clone https://github.com/Archana92166/Neuro_Adaptive_Route_Emergency_System_NARES.git
````

2. Navigate to project:

```bash
cd neuroscope
```

3. Install dependencies:

```bash
npm install
```

4. Create `.env.local` with your Firebase & Google Maps keys.
5. Start development server:

```bash
npm run dev
```

6. Open [http://localhost:9002](http://localhost:9002) in your browser.

## Project Structure

* **/src/app/** – Pages by route (e.g., `login/page.tsx`)
* **/src/components/** – Reusable UI & custom components
* **/src/lib/** – Utils, Firebase config, types
* **/src/ai/** – Generative AI flows & tools
* **/src/contexts/** – Global state providers
* **/public/** – Static assets
* `tailwind.config.ts` – Styling configuration
* `package.json` – Project dependencies
